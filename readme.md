# üìä Customer Sales Performance Dashboard - Excel-Based Data Analytics

This project delivers a highly interactive **Customer Sales Performance Dashboard** built entirely using **Microsoft Excel**. It leverages the power of dynamic charts, pivot tables, slicers, and KPI cards to provide comprehensive analysis of customer transaction data. The dashboard is designed to help businesses track essential metrics like **total revenue**, identify the **top-performing salesperson**, pinpoint **high-demand product categories**, and analyze **sales trends over time**.

---

## üéØ Dashboard Objectives

The primary goal of this dashboard is to empower business teams with actionable insights on:

* Total revenue generated over the selected period.
* The best-performing salesperson based on revenue contribution.
* The highest selling product category.
* Customer demographics influencing sales (using Gender & Age Filters).
* Month-wise and Year-over-Year sales trends.
* Category-wise distribution of quantity sold and total amount.
* Impact analysis of sales filters on Key Performance Indicators (KPIs) and graphs.

---

## üöÄ Key Features

### 1. KPI Summary Cards
A dedicated section for high-level performance indicators that update dynamically:

* **Total Amount:** Displays total revenue generated.
* **Highest Person Name:** Shows the salesperson with the maximum revenue (e.g., Bob).
* **High Product Category:** Identifies the most sold category (e.g., Beauty).
* **Total Quantity:** Sum of all purchased units.

### 2. Dynamic Timeline Filter
* Allows users to select and analyze sales data by **Year**, **Quarter**, or **Month**.
* Automatically updates all connected charts and KPI cards for seamless analysis.

### 3. Slicers for Real-Time Filtering
Users can instantly refresh the entire dashboard by filtering data using the following interactive slicers:

* `SalesPerson` (Alice, Bob, Charlie, Diana, etc.)
* `Gender`
* `Quantity`
* `Product Category`
* `Age`
